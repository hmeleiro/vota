# Ajustes manuales de ejemplo

Dataset con retoques o ajustes manuales para aplicar a las estimaciones
electorales.

## Uso

``` r
retoques
```

## Formato

Un data frame con las siguientes columnas:

- idv:

  Carácter, código del partido

- votos_adicionales:

  Entero, votos a añadir (positivo) o restar (negativo)

## Fuente

Datos sintéticos de demostración
