# Partidos pequeños de ejemplo

Dataset con estimaciones de voto para partidos pequeños no incluidos en
la matriz de transferencia principal.

## Uso

``` r
small_parties
```

## Formato

Un data frame con las siguientes columnas:

- idv:

  Carácter, código del partido

- votos:

  Entero, estimación de votos a nivel nacional

## Fuente

Datos sintéticos de demostración
