# Matriz de Transferencia de Ejemplo

Dataset de ejemplo con una matriz de transferencia electoral empleada en
las plantillas del paquete.

## Usage

``` r
mt
```

## Format

Un data frame con las siguientes columnas:

- idv:

  Factor con la intención de voto de los partidos

- PSOE:

  Numérico, porcentaje de transferencia desde recuerdo PSOE

- PP:

  Numérico, porcentaje de transferencia desde recuerdo PP

- Vox:

  Numérico, porcentaje de transferencia desde recuerdo Vox

- Sumar:

  Numérico, porcentaje de transferencia desde recuerdo Sumar

- OTBL:

  Numérico, porcentaje de transferencia desde recuerdo OTBL

- ABNL:

  Numérico, porcentaje de transferencia desde recuerdo ABNL

## Source

Datos sintéticos de demostración
