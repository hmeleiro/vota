# Número de escaños por provincia

Dataset con el número de diputados asignados a cada provincia española.

## Usage

``` r
n_seats
```

## Format

Un data frame con las siguientes columnas:

- codigo_provincia:

  Carácter, código INE de la provincia

- n_diputados:

  Entero, número de escaños asignados

## Source

Instituto Nacional de Estadística (INE)
