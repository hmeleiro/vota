# Patrones electorales del 23J

Dataset con patrones históricos de voto por provincia para las
elecciones generales del 23 de julio de 2023.

## Usage

``` r
patrones_23J
```

## Format

Un data frame con las siguientes columnas:

- codigo_provincia:

  Carácter, código INE de la provincia

- partido:

  Carácter, código del partido político

- patron:

  Numérico, proporción histórica de votos (0-1)

## Source

Ministerio del Interior, elecciones generales 23J 2023
