# Resultados electorales del 23J

Dataset con los resultados oficiales de las elecciones generales del 23
de julio de 2023, utilizados como referencia de elecciones anteriores.

## Usage

``` r
votos_23J
```

## Format

Un data frame con las siguientes columnas:

- recuerdo:

  Carácter, código del partido según recuerdo de voto

- votos_ant:

  Entero, votos obtenidos en la elección anterior

## Source

Ministerio del Interior, elecciones generales 23J 2023
